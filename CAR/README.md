# DriveWise (Project: CAR)

DriveWise is a Unity-based driving safety training simulator. The project focuses on scenario-based practice (hydroplaning, popped tire, T-bone collision avoidance, highway driving) with guided UI prompts, NPC traffic, traffic signals, and optional OpenAI-powered feedback to help drivers understand safer responses.

Unity version: `6000.0.36f1` (Unity 6)

**Input**
`Assets/InputSystem_Actions.inputactions` defines the Input System action map, including steering, throttle/brake axes, paddles, and buttons used for wheel and gamepad control.

**Rendering and Project Settings**
`Assets/Settings/` contains URP pipeline assets and volume profiles. `Packages/manifest.json` lists Unity dependencies (Input System, URP, Navigation, Visual Scripting, etc.). `ProjectSettings/` holds editor and player configuration.

**Third-Party Asset Packs**
These folders contain third-party assets for vehicles, environments, skyboxes, and props:
`Assets/PROMETEO - Car Controller`, `Assets/ARCADE - FREE Racing Car`, `Assets/AllSkyFree`, `Assets/Realistic Tree`, `Assets/kenney_city-kit-roads`, `Assets/kenney_city-kit-industrial_1.0`, `Assets/Snow mountain tracks`, `Assets/VertexModeler`, `Assets/YughuesFreeSandMaterials`, `Assets/TextMesh Pro`.

**Generated and Local Files**
`Library/`, `Logs/`, and `UserSettings/` are Unity-generated and machine-local. They are not required for source control.

**Notes on Secrets**
`Assets/Scripts/.env` contains OpenAI API keys and should be treated as a local secret file. `EnvLoader` currently looks for `StreamingAssets/.env` when loading keys.
